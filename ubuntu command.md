# 常用快捷键
  - Tab： 自动补全
  - Ctrl + A：把光标移动到开头
  - Ctrl + E：把光标移动到结尾
  - Ctrl + L：清屏

# 常见命令
  - cd <Path>
  - ls
  - pwd(Print Current Name of Directory)
  - tree
  - rm <file> / -r <document> ,-f表示强制删除，无需确认
  - 创建文件以及文件夹
    - mkdir document
    - touch file
  - cp <file> <dest_file> / -r <document><dest_document>
  - 程序卡死
    - ps -ef 查看当前所有正在运行的进程
    -kill -s9 <PID> -s9表示强制执行
